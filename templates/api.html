Credit Approval System API Endpoints

  • POST /api/register
    Register a new customer. Requires: first_name, last_name, age, monthly_income, phone_number.
  • POST /api/check-eligibility
    Check loan eligibility for a customer. Requires: customer_id, loan_amount, interest_rate, tenure.
  • POST /api/create-loan
    Create a loan for a customer. Requires: customer_id, loan_amount, interest_rate, tenure.
  • GET /api/view-loan/
    View details of a specific loan by loan ID.
  • GET /api/view-loans/
    View all loans for a specific customer by customer ID.
  • POST /api/upload-data
    Upload CSV or Excel file containing customer or loan data for bulk import.
